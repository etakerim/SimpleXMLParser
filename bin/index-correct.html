Plánovací a záznamový kalendár pre váš terminál




Jeho plánované funkcie sú nasledovné:
  • Udalosti
  • Todo
  • Kontakty
  • ..a mnoho ďalšieho rozšírenia cez templatovanie

Projekt používa rôzne opensource knižnice, aby bol ľahšie prispôsobiteľný


Sú to napríklad: Vektory (dynamické polia), UTHASH(hešovacia tabuľka) a samozrejme The better string library



Štruktúry v programe budú vyzerať následovne:

struct Event struct Kontakt
bstring title bstring name
time_t begin bstring job
time_t end bstring email[]
bstring descript bstring telnum[]
time_t edittm
bstring tags[] bstring descript

Ak chcete prispievať do tohto projektu musíte ovládať programovací jazyk C lebo naňom budujeme všetky potrebné nástroje a programy. Základný program ste preto už určite videl. Pochádza zo svetoznámej prírúčky toho jazyka: Programovací jazyk C. Autormi sú Kernighan&Ritchtie.


        
        #include <stdio.h>

        int main(void)

            printf("hello, world");
        }
